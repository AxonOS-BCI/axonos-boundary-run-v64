# Contributing

## Rules

1. No real neural data in issues, fixtures, screenshots, or tests.
2. No telemetry, analytics, beacons, trackers, or third-party CDN assets.
3. Preserve deterministic replay behavior.
4. Preserve the donation address unless the maintainer explicitly changes it.
5. Keep inbound contributions under the same license model: AGPL-3.0-only OR AxonOS Commercial License.

## Evidence discipline

Every gameplay mechanic should map to a documented AxonOS concept, RFC, or safety guarantee.
