# Governance

Boundary Run v64 follows a lightweight open-governance model.

- Public issues and pull requests are preferred.
- No CLA by default.
- Inbound = outbound license model (AGPL-3.0-only OR AxonOS Commercial License).
- Maintainer decisions prioritize safety, privacy, determinism, and traceability.
- Forkability is a first-class project property.
- All governance changes require public discussion in issues.

## Decision making

1. **Day-to-day:** Maintainer (Denis Yermakou) has final say on technical direction.
2. **Security:** Coordinated disclosure via `security@axonos.org`.
3. **Licensing:** Dual-license model is non-negotiable. No single-license conversion without community consensus.
